# Classify Assist AI Placement Plugin for Moodle

## Description

The **Classify Assist AI Placement Plugin for Moodle** is an AI Placement plugin that uses a configured AI Provider to assign competencies to Moodle activities and courses. The plugin is tested primarily with Ollama and AWS Bedrock.

## Requirements

1. Moodle 5.x or above.
2. AI Provider configured and operational.
